6. Setup & Deployment Guidelines

To run the production Node.js server and connect the local client:

  1. Ensure you are in the project folder: rpsc-ras-app
  2. Set environment variables:
    $env:TURSO_DATABASE_URL="your-url" (PowerShell)
  3. Start the backend node server:
    node backend/server.js
